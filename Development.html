Trader Joe's Holiday Picks with React

For this project, I was required to develop an interactive interface that contains a list of item cards, 2 filtering categories, 1 sorting feature, and an aggregator section. Based on the requirements, I came up with the idea of creating a Trader Joe's customer check out page, as the company currently does not have an online ordering platform for its customers. This is, undoubtedly, a challenging project to implement given the complexity of JavaScript and React.
